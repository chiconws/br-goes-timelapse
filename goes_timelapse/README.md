# App GOES Timelapse

App do Home Assistant para acompanhar municípios brasileiros com imagens do GOES-19.

- usa a Banda 2 visível do GOES-19
- baixa os dados oficiais da NOAA
- converte os quadros para um cache local otimizado
- gera animações WebP na mídia do Home Assistant
- oferece interface Ingress para busca e acompanhamento dos municípios
